School Classification Import/Delete

Step 1

At the beginning of each ranking season you will want to import the newest classifications. To start that process you will need to delete the last seasons classifications. Please click the delete button to remove the current classifications

Use this with caution to remove last seasons classification and import a new classification

Step 2

You will need to format the classification list obtained from the Iowa High School Athletics Association site (iahsaa.org) by using this template. You should be able to use excel or Google spreadsheets to edit the CSV template. Do NOT remove the headers in this template.

Step 3

Click the upload button and choose the file you just formatted

That's it! You have now imported the schools. You can navigate to the school listing to see what you imported.
